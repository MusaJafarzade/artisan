# VFS Smart Login System

Система автоматизации входа на платформу VFS Global, предназначенная для оптимизации процесса авторизации и обхода систем защиты Cloudflare.

## Основной стек
- **Python / Selenium**: Использование WebDriver для имитации поведения реального пользователя.
- **2Captcha API**: Автоматическая интеграция для решения капч в режиме реального времени.
- **Stealth Techniques**: Применение техник маскировки браузера для предотвращения блокировок.

## Функционал
- Автоматический ввод учетных данных.
- Обработка капчи через API.
- Устойчивость к базовым защитным механизмам сайта.

## Статус
Проект находится в стадии активной разработки (Proof of Concept).
